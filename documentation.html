Website Documentation

Bootstrap Components Used

The following Bootstrap components are included in this website:

  • Navbar (responsive, collapsible)
  • Buttons (primary, secondary, light)
  • Modals (contact form and grade modals)
  • Accordion (Skills section)
  • Carousel (Visualisations section)
  • Dropdown (Grades selection in Education section)
  • Cards (Work Experience, Volunteering)
  • Grid system (rows and columns for layout)
  • Container, Container-fluid, spacing utilities (padding, margin)
  • Text utilities (text-center, text-white, text-muted, lead)
  • Badge (used in commented-out section)

Dynamic Features

The following interactive/dynamic features are included in this website:

  • Modal open/close (Bootstrap JS)
  • Accordion expand/collapse (Skills section)
  • Carousel slide with next/previous controls (Visualisations section)
  • Dropdown toggle for Grades selection
  • Responsive Navbar toggle (collapsible menu on smaller screens)
  • Smooth scrolling for internal page links (CSS: scroll-behavior: smooth)
  • Form textarea (contact modal, interactive but not submitted anywhere)
